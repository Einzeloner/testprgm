git config --global user.name "username"
git config --global user.email "email"

git init  = initalizes a repository in the specified folder.


git status = displays status of the repo


git add filename = adds the file to the staging area


git commit -m "your message" = commits with a message


git log = gives you a history of commits



git checkout <hashname> = helps you go back to one desired log



git branch <new branch name> = helps to create a new branch instead on the main branch or timeline


git checkout <branch name> =  puts you into that branch
